# Changelog

## v1.0.0 — Initial Release

- 5-stage automated lead generation pipeline (Discovery, Qualification, Personalization, Outreach, Reporting)
- Web dashboard for status monitoring and manual pipeline control
- SQLite database — no external DB required
- Google Places API integration for business discovery
- Groq AI-powered email personalization and PDF proposals
- Brevo SMTP email dispatch with retry logic
- APScheduler for automated daily pipeline execution
- Optional Telegram notifications
- Docker and docker-compose support
- All API keys provided by user — zero platform cost
